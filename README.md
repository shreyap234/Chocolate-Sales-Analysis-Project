# 🍫 Chocolate Sales Analysis Dashboard

## 📌 Project Overview
The **Chocolate Sales Analysis Dashboard** is an interactive Power BI project developed to analyze chocolate sales performance across different regions, years, categories, products, and order statuses.

This dashboard helps in understanding business performance through KPIs, charts, and filters, making it easier to identify sales trends and product contribution.


## 🎯 Objective
The objective of this project is to convert raw chocolate sales data into meaningful insights for better business decisions, product planning, and sales monitoring.

## 🛠️ Tools & Technologies Used
- **Power BI**
- **Excel / CSV Dataset**
- **Data Cleaning**
- **Data Visualization**
- **DAX Measures**
- **Slicers / Filters**

## 📈 Dashboard Features
- Region-wise filtering
- Year-wise analysis
- Category-wise sales breakdown
- Order Status analysis
- Product performance tracking
- Monthly sales trend
- Geo-wise sales distribution
- Boxes sold by category

---

## 🔍 Key Insights
- **Bars** category contributed the highest amount among all categories.
- **UK and India** showed the highest count of amount by geography.
- Sales fluctuated across months, with peak performance in selected months.
- **Raspberry**, **85% Dark**, and **Spicy Special** were among the top-performing products.
- Total order volume and total amount indicate strong product demand.
